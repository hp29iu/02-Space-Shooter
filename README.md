# 02-2D-Space-Shooter
<Project 2 - 2D Shooter Game> InToTheUnknown, 25 September 2021

Please, Go ahead and defeat all your enemies with your weapons to save the unknown world.

Forward - up arrow
Back - down arrow
Left - left arrow
Right - right arrow

Weapon 1 - Q
Weapon 2 - W
Weapon 3 - E
Weapon 4 - R

Pause - P
Quit - ESC


## Implementation
Built using Godot 3.3.3
